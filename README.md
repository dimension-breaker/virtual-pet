### Run Your Own Instance
1. Download ```Node.js```.
2. Download this project.
3. Navigate to the project directory.
4. Type ```npm install -g @angular/cli```.
5. Type ```npm install```.
6. Type ```npm start```.
7. To build online, follow these wonderful instructions: https://angular.io/guide/deployment
